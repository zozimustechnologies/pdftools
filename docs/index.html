Everything you need. Nothing you don’t.

A complete PDF toolkit in a compact sidebar — zero uploads, zero accounts, zero complexity.

🔒

Password Protection

Encrypt PDFs with AES-256. Set separate user and owner passwords, and control printing, copying, editing, annotations, and form filling independently.

🔓

Remove Password

Decrypt any PDF you own. Enter the password once and download a clean, unencrypted copy instantly.

✏️

Digital Signatures

Draw your signature directly in the side panel using mouse or touch. Attach name, reason, page number, and an optional timestamp.

💧

Watermarks

Add custom text watermarks with full control over placement, opacity, font size, and colour. Live preview updates as you type.

✂️

Extract Pages

Pull out specific pages by number or range (e.g. 1–3, 5, 7) and download them as a new PDF in seconds.

🔀

Merge PDFs

Combine multiple PDF files into a single document. Load the first PDF, add more, and merge with one click.

📑

Split PDF

Separate every page of a PDF into its own individual file. Download all pages in one go.

↻

Rotate Pages

Rotate pages 90°, 180°, or 270° clockwise or counter-clockwise. Works on the entire document at once.

📄

Edit Metadata

Update the document’s title, author, subject, keywords, creator, and producer fields without re-creating the file.

Get started in seconds

Install the extension

Get it from the Edge Add-ons store.

Open the side panel

Click the PDF Tools icon in your Edge toolbar. The panel opens alongside any page.

Load a PDF

Click the file picker to choose a PDF from your device, or use “Use Current Tab” if the tab has a PDF open.

Process & download

Pick any tool, configure it, and click the action button. Your processed PDF downloads automatically.
